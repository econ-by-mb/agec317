# agec317: Economic Analysis for Agribusiness Management

Welcome! This is a repository containing all relevant materials for AGEC 317, as taught by [Michael Black](https://michael-black.github.io) in the Spring of 2020 at [Texas A\&M University](https://www.tamu.edu). Browse through the repository for all materials, or see below for the syllabus and lecture notes.

Feel free to clone the repo and use the materials for yourself. If you are using any of the material in your own course, please [let me know](mailto:black.michael@tamu.edu), and if you could cite this repo or my [personal site](https://michael-black.github.io), that would be great!
